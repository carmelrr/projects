# Feature List by Role and Phase

## Phase Definitions

- **MVP** — minimum to deliver real coaching value
- **Growth** — deeper engagement, retention, and feedback tools
- **Scale** — team management, third-party integrations, advanced automation

---

## Coach Features

### MVP

**Client Management**
- Create client profiles with email invite
- View client list with filters (status, compliance, last activity)
- View client profile (goals, notes, history)
- Archive / pause / restore clients
- Assign coach to client

**Program & Workout Builder**
- Create multi-week program templates
- Drag-and-drop workout calendar builder
- Add workouts to specific days
- Set per-workout type (Strength, HIIT, AMRAP, EMOM, For Time, Cardio, Mobility)
- Add exercises from library with inline video preview
- Define sets, reps, weight targets, tempo, rest, and notes per exercise
- Group exercises into supersets/circuits (A1, A2, B1...)
- Duplicate workouts and programs
- Save workouts as reusable templates
- Assign program to one or multiple clients
- Schedule individual ad-hoc workouts to a client's calendar

**Exercise Library**
- Search system exercise library (3000+ exercises with video)
- Create custom exercises with video/photo upload
- Organize exercises by category, muscle group, equipment
- Add and replace exercise demo videos

**Messaging**
- 1:1 chat with each client
- Send text, images, GIFs
- Read receipts + typing indicators

**Notifications**
- Notification when client submits workout log
- Notification when client sends a message

**Basic Metrics**
- Create custom metric definitions (name, unit)
- View client metric history in a graph
- Log metric entries manually on behalf of client

**Web Dashboard**
- One-screen "Today" overview: who trained, who missed, who needs attention
- Weekly compliance overview per client

---

### Growth

**Compliance & Alerts**
- 7/30/90-day compliance percentage per client
- "Needs Attention" flag when compliance drops 20%+
- Color-coded compliance status on client list

**Voice Notes**
- Record and send voice note messages
- Attach voice notes to specific workout logs
- Listen to client voice note check-ins

**Video Feedback**
- View client-uploaded workout videos in log context
- Annotate/respond with text + voice note

**Check-ins & Assessments**
- Create assessment templates (movement screens, check-ins)
- Assign assessment to client
- Score each item + add notes + attach video
- Mark assessment as reviewed

**Habit Tracking**
- Define habits (daily/weekday/custom schedule)
- Mark habits as counting toward compliance score
- View client's habit completion history

**Nutrition Review**
- Set client macro/calorie goals
- View client's daily nutrition entries
- View subjective markers (energy, stress, sleep, steps)

**Group Messaging**
- Create group message threads (challenges, cohorts)
- Broadcast message to multiple clients

**Progress Photos & Metrics**
- View client-uploaded progress photos with date
- Side-by-side comparison view

---

### Scale

**Team Management**
- Invite other coaches (Admin Coach role)
- Assign clients to specific coaches
- View per-coach client load
- Transfer clients between coaches
- Org-wide shared exercise and program library

**Automation / Webhooks**
- Trigger outbound webhooks on events (log submitted, compliance drop, etc.)
- Zapier integration

**Wearable Data View**
- View synced data from Apple Health, Garmin, WHOOP (if client connected)
- Overlay wearable metrics with workout calendar

**Advanced Analytics**
- Coach-level dashboard: team compliance, retention trends
- Client retention metrics
- Activity heatmaps

---

## Client / Athlete Features

### MVP

**Onboarding**
- Accept invite link → create account
- Complete profile (name, goals, photo)
- View assigned coach

**Workout Experience**
- View today's workout in calendar strip
- Full monthly calendar view
- Tap workout to see: instructions, exercises, sets/reps targets
- Watch exercise demo video per exercise
- Log each set (reps, weight, time, distance)
- Log RPE (1-10) and overall workout notes
- Finish workout → status changes to Completed
- Reschedule workout to another day (self-service)

**Offline Resilience**
- Log workout fully offline (no connection required)
- Auto-sync when connection restored

**Messaging**
- Chat with coach (text, images, GIFs)

**Notifications**
- Push notification when new workout assigned
- Push notification when coach sends message or feedback

---

### Growth

**Video Upload**
- Upload workout videos from camera roll
- Record and upload a set/movement from within the workout log screen

**Voice Notes**
- Record voice note check-in and send to coach
- Listen to coach voice note feedback

**Habit Check-in**
- View daily habit list
- Check off habits (binary or quantitative)
- View streak/history

**Nutrition Logging**
- Enter daily macros manually (calories, protein, carbs, fat)
- View goal vs actual
- Log subjective markers (energy, sleep, water, steps, stress)

**Progress Tracking**
- Log body metrics (weight, measurements) via metric entries
- View metric history as graphs
- Upload progress photos

**Assessment Participation**
- Receive assessment assignment
- Complete each item (with video/photo upload)
- View coach feedback and scores

---

### Scale

**Wearable Connection**
- Connect Apple Health / Garmin / WHOOP
- Auto-sync daily activity, heart rate, sleep
- View synced data alongside workout calendar

---

## Admin Features

### MVP

**User Management**
- View all users in organization
- Suspend or reactivate users
- Reset user access

**Audit Log**
- View log of sensitive actions (archive client, role change, data export)

**System Settings**
- Org timezone, name, branding

---

### Growth

**Data Export**
- Export client data (GDPR-compliant)
- Export audit logs

### Scale

**Advanced RBAC**
- Custom role configurations
- Fine-grained permissions per coach
